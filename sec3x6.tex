\]
となることである．

\証明

必要性．$P_0,p_1, \cdots, p_r$が$K$の1つの単体$	\sigma$の頂点であるならば，
各$	i (0 \leqq \leqq r)$に対して，$	\text{Int } \subset O(p_i)$が成り立ち，
$ O(p_0) \cap O(p_1) \cap \cdots \cap O(p_r) \supset \text{Int } \sigma \ne \varnothing$

$ \text{Int } \subset O( \varphi	(	p_i) ; L)$となり，
$ \varphi	(p_i) \in \tau$を得る．

\定理{14}

\subsubsection{第3章の問題}

1. $\R^n$の点$v_0, v_1 , \cdots , v_r$が一般の位置にあるための必要十分条件は，
$	\sum_{i=0}^{r}	\lambda_i v_i = 0,	\sum_{i=0}^{r} \lambda_i =0$，
$	\lambda_i \in \R ( 0 \leqq i \leqq r)$ならば，$	\lambda_0 = \lambda_1 = \cdots = \lambda_r = 0$
が成り立つことである.

\証明
必要性．$0 = \sum_{i=0}^{r}	\lambda_i v_i =(-\lambda_1 - \lambda_2 - \cdots - \lambda_r) v_0 + \lambda_1 v_1 + \cdots + \lambda_r v_r
= \sum_{i=1}^{r} \lambda_i(v_i - v_0)$．$v_0, \cdots , v_r$が一般の位置にあることから，
$\lambda_1 = \lambda_2 = \cdots = \lambda_r =0 = \lambda_0$．

十分性．$\sum_{i=1}^{r}	r_i ( v_i - v_0) = 0 \: ( a_i \in \R )$と仮定すると，
$(-a_1 -a_2 - \cdots - r_r)v_0 + a_1 v_1 + \cdots + a_r v_r =0$で係数の和$=0$であるから，
$a_1 = a_2 = \cdots = a_r =0$となり，$v_0 , v_1 , \cdots , v_r$は一般の位置にある．

$ \sigma$の1つの頂点$ p$を任意にとって，$\pi ( b_\sigma = p$と定めて，
写像$ \pi : Sd K^{(0)} \to K^{(0)}$を定義すると，
$ \pi : Sd K \to K$は単体写像となる．更に，$\pi$は恒等写像
$ \text{id } : | Sd K | \to |K|$の単体近似である．

\証明

$\text{Sd} K $の$k$単体$|b_{\sigma_0} b_{\sigma_1} \cdots b_{\sigma_k}|
\: ( \sigma_0 < \sigma_1 < \cdots < \sigma_k, \sigma_i \in K )
$に対し，$	\pi ( b_{\sigma_0}), \pi ( b_{\sigma_1}, \cdots , \pi ( b_{\sigma_k} )$
はいずれも$	\sigma_k$の頂点であるから，$\pi$は単体写像である．
次に，$	\sigma = | p_0 p_1 \cdots p_r |$を$K$の任意の$r$単体とすると，
$b_\sigma$は$\text{Sd} K$の1つの頂点である．次に，$\sigma = | p_0 p_1 \cdots p_r |$
を$K$の任意の$r$単体とすると，$b_\sigma$は$\text{Sd} K$の1つの頂点である．
$	b_\sigma$を頂点としてもつ$\text{Sd } K$の単体を$\tau = | b_{\sigma_0} b_{\sigma_1} \cdots
b_\sigma \cdots b_{\sigma_k} | $ （$\sigma_0 < \sigma_1 < \cdots < \sigma \cdots < \sigma_k$）
とすると，
$\text{Int} \tau \subset O ( p_i ; K)$である．なぜならば，$\text{Int} \tau$の
任意の点を重心座標で$	\lambda_0 b0 + \cdots + \lambda b_\sigma + \cdots + \lambda_k b_{\sigma_k}$
と表したとき，$b_\sigma = \sum_{i=0}^{r} (r+1)^{-1} p_i$
であるから，$p_i$の係数 $ \geqq (r+1)^{-1} \lambda > 0$となるからである．
ゆえに，$O ( b_\sigma ; \text{Sd} K) \subset O ( \pi (b_\sigma) ; K)$が成り立つ．
これは恒等写像 $ \text{id } : |\text{Sd } \to | K |$の単体近似であることを意味する．
